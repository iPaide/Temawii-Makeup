# Temawii Makeup Demo

Static outreach website preview for Temawii Makeup.

## Deploy on Render

1. Create a new GitHub, GitLab, or Bitbucket repository.
2. Upload the contents of this folder as the repository root.
3. In Render, choose **New +** > **Blueprint**.
4. Select the repository.
5. Render will detect `render.yaml`.
6. Click **Apply**.

Render settings:

- Service type: Static Site
- Build command: empty
- Publish directory: `.`

Before sharing publicly, replace:

- Placeholder WhatsApp number in `index.html`
- Temporary generated images in `assets/`
- Example review themes with real testimonials
- Any award, pricing, policy, and service details not confirmed by Temawii Makeup
